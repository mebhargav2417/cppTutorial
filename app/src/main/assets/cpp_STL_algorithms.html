Algorithms

     Algorithm that do something to the element in container. These algorithms are not member functions or even friends of container classes, these are standalone template functions. We must include <algorithm> in our program to access STL algorithms. Let us see some of algorithms listed below,

Algorithm Purpose
count Counts the number of elements in a container
find Finds the first occurance of element in a container
equal Compares the contents of two containers and returns true if all corresponding elements are equal
search Searching for a sequence of values in one container that corresponds with the same sequence in another container
copy Copies a sequence of values from one container to another
remove Removes specified elements
replace Replace elements with a specified element or value
swap Swaps or exchanges elements
sort Sorts the container values according to a specified order
reverse Reverse the order of elements
merge Merge two sorted containers
