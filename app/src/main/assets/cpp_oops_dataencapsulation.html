Classes

     Classes are nothing but user defined data type, having its own data members and member functions. We will access those using instance of class also known as objects. Variables and functions which are defined and implemented for classes are called data members and member function.
Let us consider a class called Rectangle which helps to draw rectangle shape and to calculate of its area. Rectangle have length and breadth, nothing but variables for class rectangle. With the help of draw and area functions we will get shape of rectangle and its area. Those area and draw are member functions for class rectangle.

Class syntax is as follows

class <Class name>{
     Access Specifier:
     Data members;
     Member functions;
};

Example for classes is as follows:

class Rectangle {
     public:
     int length, breadth;
     public:
     void draw(){
     //definition of draw function
     }
     void area();
     //definition of area function
     }
};

From the above example, public is an access specifier or visibility control. With help of this access specifier we will perform data hiding. Nothing but we will control the access of data members and member functions from outside of class environment. Data members are length and breadth of int data type. Draw and area are class member functions.

Objects

     Object is an instance of class. We can access data members and member functions of class using object. An object is also called class data type.
