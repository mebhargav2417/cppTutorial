Standard Template Library

     Alexander Stepanov and Meng Lee of Hewlett Packard developed a set of general purpose template classes (data structures) and functions (algorithms) helps to storing and processing of data in standard approach. The collection of the generic classes and functions called STL (Standard Template Library).

STL components are :

  1. Containers
  2. Algorithms
  3. Iterators
Containers helps to store data in organized way(data structures) like arrays, list, queue and so on. Also these are generic too, they can hold elements of any data type, for example: vectors can be used to create dynamic arrays of char, integer, string and other.

Algorithms helps to process the data in the containers. STL have many different types of algorithms to provide support for tasks such as searching, sorting, copying and merging.

Iterators are like pointers used to access elements in the container. Used to traverse from one element to another(nothing but iterating through the container).

STL components which are now part of the Standard C++ Library are defined in the namespace std. So we must use the directive as follows

using namespace std;

it informs to compiler that we intend to use the Standard C++ Library.
